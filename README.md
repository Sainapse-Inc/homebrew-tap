# Homebrew tap

Homebrew formulas for projects by [Abhirup Ghosh](https://github.com/abhirupghosh).

## repobox

After the first signed release is published:

```sh
brew tap abhirupghosh/tap
brew install repobox
```

The repobox release workflow generates `Formula/repobox.rb` from the release archives and their SHA-256 checksums.
