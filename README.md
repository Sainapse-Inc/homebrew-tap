# Homebrew tap

Homebrew formulas for projects maintained by
[Sainapse](https://github.com/Sainapse-Inc).

## repobox

After the first signed release is published:

```sh
brew tap Sainapse-Inc/tap
brew install repobox
```

The [Repobox release workflow](https://github.com/Sainapse-Inc/repobox/actions/workflows/release.yml)
generates `Formula/repobox.rb` from the release archives and their SHA-256
checksums.
